# 📶 Smart Network Analyzer Chrome Extension

## 🚀 Description
A Chrome extension that monitors network performance in near real-time. It detects issues like slow speed, high latency, and possible network congestion, and provides actionable suggestions along with a network health score.

## 🔍 Features
- Real-time network monitoring
- Detects high latency and slow speed
- Identifies possible network congestion
- Provides smart suggestions
- Displays Network Health Score

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extension APIs


## ▶️ How to Use
1. Download this repository
2. Open Chrome → chrome://extensions/
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the project folder
